# Retrieval examples — Homework #2

Generated by `scripts/run_test_queries.py --k 3`.
Embedding model: `text-embedding-3-small` (chunks and queries encoded with the same model).
Vector store: Chroma, cosine distance. **Score = `1 - cosine_distance`**, so 1.000 is
identical and 0.000 is orthogonal.

The query set is deliberately mixed: direct queries that reuse the corpus's vocabulary,
paraphrased queries that avoid it, cross-document queries, and one out-of-corpus query.
A set of only direct queries would score uniformly high and prove nothing — see the
analysis at the end.
